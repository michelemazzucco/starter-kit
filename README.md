# Starter kit [![Dependency Status][ds-image]][ds]
Super easy static website starter kit with PostCSS.

PostCSS plugins installed:
* [Autoprefixer]
* [Lost Grid]
* [Rucksack]
* [CSSnano]

## Getting started
```
$ npm install --global gulp
$ npm install
$ npm start
```

[Lost Grid]:    https://github.com/peterramsing/lost
